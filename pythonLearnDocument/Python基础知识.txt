3.列表和元组
    + 列表可以修改而元组不可以。
    + 列表所有元素放在中括号内，user = ['tom',18]
    + 列表和元组是序列
    + 通用的序列操作
        - 索引 序列中所有的元素都有编号，从0开始递增，可以只用索引来获取序列中的元素，使用负数时即从最后一个向前数，-1是最后一个元素的位置
        - 切片 索引可以访问单个元素，使用切片可以访问范围内的元素，中括号内使用冒号分隔两个索引，第一个表示开始的位置，第二个表示结束的位置，
        左闭右开，如果第二个索引比第一个小则返回空系列，第二个索引为空表示最后一个位置，第一个索引为空表示第一个元素的索引，在添加一个冒号表示步长
        步长不能为0但是可以为负数，为负数时表示从后边向前取元素，此时需要第一个索引大于第二个索引
        - 序列相加 可以直接用加法运算符拼接序列，一般而言不能拼接不同类型的序列
        -乘法 将序列与数字相乘时，将重复这个序列x次来创建一个新序列
        - 成员资格 in可以检查特定的值是否包含在序列中
    + 列表
        - 可以将任何序列使用list函数转换为列表
        - 修改类表 可以直接使用索引修改列表
        - 删除元素 del 列表名[索引]
        - 切片赋值 可以赋值长度与原来不同的元素，可以插入元素，可以删除元素
        - 列表方法
            * append 在列表末尾追加元素
            * clear 青龙湖列表的内容
            * copy 常规的赋值，只是将另一个名称关联到列表，copy复制出了一个新列表
            * count 统计指定元素在列表中出现了多少次
            * extend 可以将多个值附加到列表的末尾
            * index 方法index在列表中查找制定第一次出现的索引
            * insert 将一个对象插入列表
            * pop 从列表的末尾返回并删除这一元素
            * remove 删除第一个指定值的元素
            * reverse 按相反的顺序排列列表中的元素
            * sort 用于就地对列表进行排序，可以接受两个参数，key表示按照key来排序，reverse表示是否倒序
    + 元组
        - 小括号与逗号组成元组
        - 元组除了不能修改其他与列表类似，只有一个值得元组需要在元素后面加逗号
